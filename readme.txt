== Description ==

Logs the most recent successful login for each user, as well as all unsuccessful logins.

== Installation ==

1. Upload the login-logger folder to your /wp-content/plugins/ directory
2. Activate the plugin

== Frequently Asked Questions ==

= I can't find the logs in the Settings menu =

The login logs are stored in the Users menu, not the Settings menu.
